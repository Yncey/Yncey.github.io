Ubuntu保存git用户名和密码

发表于2020-12-11|更新于2020-12-11|linux
|阅读量:

如何在Ubuntu下解决Git保存用户名和密码呢？

打开Git 配置文件,该文件中保存着所有的配置信息

修改配置文件，添加下面这一行。

记住密码

helper = store #用来保存密码,提交一次会自动记住

密码保存一个小时

helper = cache – timeout 3600 # 密码只保存一个小时
